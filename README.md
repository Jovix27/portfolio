# Joseva A - Portfolio

Civil Engineering undergraduate specializing in BIM, AI-driven construction, and sustainable infrastructure design.

## Portfolio Website

Visit the live portfolio at: [https://jovix27.github.io/portfolio/](https://jovix27.github.io/portfolio/)

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- Shadcn UI

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This portfolio is automatically deployed to GitHub Pages using GitHub Actions. The deployment workflow:
1. Builds the project with the correct base path
2. Verifies the build output
3. Deploys to GitHub Pages
